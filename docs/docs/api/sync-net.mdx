# SyncNet

同步 HTTP 客户端，支持 `with` 上下文管理。

接口与 [Net](/api/net) 完全一致，区别仅在于：

- 所有方法为同步调用（无 `async`/`await`）
- 使用 `with` 而非 `async with`
- 使用线程信号量（`threading.Semaphore`）控制并发
- 不支持 `aiohttp` 引擎
- 信号中间件仅支持同步回调

## 构造函数

```python
SyncNet(
    engine: str | EngineEnum | type[SyncEngineBase] = None,
    *,
    base_url: str = None,
    timeout: float = None,
    retries: int = None,
    retry_delay: float = None,
    user_agent: str = None,
    proxy: str | ProxyService = None,
    verify: bool = None,
    raise_status: bool = None,
    allow_redirects: bool = None,
    rate_limit: int | float | RateLimitConfig = None,
    concurrency: int = None,
    headers: dict[str, Any] = None,
    cookies: dict[str, Any] = None,
    engine_options: dict[str, Any] = None,
    config: NetConfig = None,
)
```

参数同 [Net](/api/net#构造函数)。


### stream

```python
def stream(method: str, url: str, **kwargs) -> StreamResponse
```

发起流式请求，返回 [`StreamResponse`](/api/stream-response) 对象。使用 `with` 上下文管理器确保资源释放。

```python title="sync_stream.py"
with SyncNet() as net:
    with net.stream("GET", "https://example.com/large-file") as resp:
        with open("file.zip", "wb") as f:
            for chunk in resp:
                f.write(chunk)
```

## 属性

### cookies

```python
@property
def cookies(self) -> dict[str, str]
```

获取当前会话的 cookies。

### proxy_service

```python
@property
def proxy_service(self) -> ProxyService | None
```

获取代理服务实例，可用于调用 `switch()` 切换代理。

## 可用引擎

| 引擎 | 支持 |
|------|------|
| httpx | ✅ |
| curl-cffi | ✅ |
| requests | ✅ |
| requests-go | ✅ |
| aiohttp | ❌ |

## 使用示例

```python title="sync_example.py"
from hs_net import SyncNet

with SyncNet(engine="requests", retries=3) as net:
    resp = net.get("https://example.com")
    print(resp.css("title::text").get())
```
